# Contact and Support

## How Users Receive Notifications

Users on Hungry Saver receive notifications through multiple channels:

- Email notifications for important updates, status changes, and platform communications
- In-app notifications visible in user dashboards
- Real-time updates when tasks are accepted, picked up, or delivered
- Status change notifications for donations and requests

Email notifications are sent for:
- Volunteer application approval or status changes
- Donation task acceptance by volunteers
- Task status updates
- Request approval or status changes
- Important platform announcements

Users can view their notification history and current status updates through their respective dashboards.

## Email Communication via SendGrid

Hungry Saver uses SendGrid for email delivery. The platform sends automated emails for:

- Account verification and password resets
- Volunteer application status updates
- Donation and request status notifications
- Important platform communications
- Support and help responses

All emails are sent from the platform's official email address and include relevant information about the user's account, tasks, or requests. Users should check their email regularly for important updates and ensure their email address is current in their profile.

## How to Get Help

Users can get help through several channels:

- Chatbot: The platform includes a chatbot feature accessible to donors, volunteers, and community support users. The chatbot provides answers to frequently asked questions and can help users understand platform features and processes.

- Dashboard Help Sections: Each user dashboard includes help sections with information specific to that user role.

- Email Support: Users can contact platform support via email for assistance with account issues, technical problems, or questions about using the platform.

- In-App Support: Users can access support information and contact options directly through their dashboards.

The chatbot is available 24/7 and provides instant answers to common questions. For more complex issues or specific concerns, users should contact support via email.

## What to Do if Issues Occur

If users encounter issues on the platform, they should:

1. Check the FAQ section or chatbot for answers to common questions

2. Review their dashboard for any status updates or notifications that might explain the issue

3. Contact platform support via email with:
   - A clear description of the issue
   - Relevant account information
   - Screenshots or details about when the issue occurred
   - Any error messages received

4. For urgent issues, such as safety concerns or delivery problems, contact support immediately

5. For disputes between users, contact platform administrators who can help resolve conflicts

The platform is committed to addressing user concerns promptly and ensuring a positive experience for all participants. Support staff work to resolve issues efficiently while maintaining the platform's values of dignity, transparency, and community support.
